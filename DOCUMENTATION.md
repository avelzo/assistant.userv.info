# Documentation d'utilisation

## Paiement en mode test

Utilisez les cartes ci-dessous pour tester l'achat de packs de crédits via Stripe.

Vocabulaire utilisé dans l'application :
- `compte` : votre profil utilisateur associé à une adresse email
- `crédits` : le solde utilisé pour générer des courriers après l'essai gratuit
- `paiement` : l'achat Stripe qui ajoute des crédits à votre compte

Conseils de test :
- utilisez de préférence l'adresse email de votre compte au moment du paiement
- après le retour Stripe, les crédits sont ajoutés au compte et visibles dans l'interface
- utilisez une date future et un CVC valide pour la carte de test choisie
